Common Questions

Q: Can I use the timer for multiple feats?  
A: Yes, but it's designed for one feat per session. Better to do separate timers for each feat.

Q: What if I finish before the timer?  
A: Great! Mark feat complete, let timer finish, then take your break early.

Q: Can I extend the timer mid-session?  
A: No. If you need more time, let it go into overtime, or start a new session after a break.

Q: Does using the timer give extra points?  
A: No, but it counts toward Focus Master achievement (10 sessions).

Q: Can I use an external timer instead?  
A: Yes! But built-in timer tracks toward achievements.

Q: What if timer stresses me out?  
A: Try shorter durations (10-15 min) or skip the timer. It's a tool, not a requirement.
