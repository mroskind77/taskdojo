Handling Difficult Situations

Disciple Lies About Completion

What you see:
Marked "Clean room" complete
Room is not clean
Claim they did homework, but didn't

How to handle:

Stay calm - Don't react emotionally
Verify facts - Make sure you're right
Request revision kindly:
   > "I'm seeing this marked complete, but when I checked, it wasn't done yet. Can you go finish it now? I'll approve once it's done!"

If pattern continues - Have serious conversation:
   > "I've noticed several times you've marked things complete that weren't actually finished. Help me understand what's going on. Are the feats too hard? Do you need help?"

Explain consequences - Not punishment, but reality:
   > "When you mark things complete that aren't done, you're only cheating yourself. The points don't mean anything if you didn't earn them. Task Dojo is here to help you build real skills."

Consider if fit - Is Task Dojo right for this disciple right now?

---

Consistently Poor Quality

What you see:
Multiple Poor ratings in a row
Struggling with all feats
Frustration building

How to handle:

Check feat difficulty - Are feats too hard?
Reduce volume - Too many feats?
Use tools - Teach AI Breakdown, Focus Timer
Have conversation:
   > "I've noticed you're struggling with feats lately. What's making it hard? Let's problem-solve together."

Adjust support - May need more scaffolding
Celebrate effort - Even Poor quality deserves recognition for trying
